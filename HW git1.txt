 Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
Сначала копируем ссылку на репозиторий с git hub, далее командой git clone <ссылка на репозиторий> 

Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?
Нужно перейти в проект, из которого хотим создать git репозиторий и воспользоваться командой git init. Далее git add или git add . (для всех файлов), чтобы добавить необходимые файлы. Потом  git commit -m  - для создания комета с комментарием и затем git push.
	
Как получить на локальный компьютер изменения с github.com
С помощью команды  - git pull или git fetch


Какую информацию нам дают команда :
git status - команда для просмотра статуса файлов, покажет в каких файлах были произведены изменения, новые и удалённые, или требующие добавления
git log - история коммитов, причисляет коммиты с информацией: уникальный ключ, кто создал, когда, эл почта автора и сообщение, что было сделано 
git remote -v - посмотреть адреса наших удаленных подключений к другим репозиториям
